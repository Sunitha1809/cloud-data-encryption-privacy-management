Cloud Data Encryption: Key Management & Data Privacy

A secure cloud file management system that allows users to:
• Create accounts securely
• Upload files
• Encrypt files using AES
• Store encrypted files in cloud storage
• Download and decrypt files using a secret key

• AES Encryption
